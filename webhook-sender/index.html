  • Discord emojis (normal): Use format <:emoji_name:emoji_id>

To get Discord emoji ID:

  1. In Discord, type \:emoji_name: and send it
  2. Discord will show the full format with the ID
  3. Copy that entire thing into your message

Example: <:pepe:123456789012345678> or <a:dancing:987654321098765432>
